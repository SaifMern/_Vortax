# VortaxStudio Production Website

A professional, responsive and scalable Vite + React + Tailwind CSS website for VortaxStudio.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Images

Logo path:

```txt
public/images/logo.png
```

Team image placeholders. Add your real images with these names:

```txt
public/images/team-1.jpg
public/images/team-2.jpg
public/images/team-3.jpg
```
